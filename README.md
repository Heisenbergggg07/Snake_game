Anaconda Snake Game

A Python-based Snake game built using Pygame with a focus on clean design, smooth gameplay, and interactive user experience.

-> Features

-Start screen and game over screen

-Dynamic difficulty (speed increases with score)

-Sound effects for actions

-High score tracking

-Custom visuals and grid system

-> Tech Stack

-Python

-Pygame

-> How to Run

-Install pygame:

pip install pygame

-Run the game:

python game.py
